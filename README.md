## FAQ

### shared proxy란 무엇입니까?
Shared Proxies는 여러 사용자가 공유하는 IP입니다. IP 독점이 필요하지 않은 경우 비용 효율적인 선택지입니다.

### shared proxies를 여러 セッション에 사용할 수 있습니까?
네, セッション 유지 기능을 사용하여 동일한 shared proxy를 여러 セッション에 사용하실 수 있습니다.
